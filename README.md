## Installation

Install my-project with npm (node.js)

#*1 PERCORSO FILE*

```bash
  cd GreenStationMonitoring
```

#*2 INSTALL DEPENDENCIES*

```bash
  npm install 
```

#*3 BUILD IL PROGETTO*

```bash
  npm install 
```

#*4 RUN NEL BROWSER*

```bash
  npm run dev
```

#*5 INDIRIZZO PER LA PREVIEW*
```bash
  *go to localhost:5173 or 8800 / 8000*
```
